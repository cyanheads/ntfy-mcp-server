# 1.0.0 — 2025-03-27

## Added

- **`send_ntfy` tool** — publish notifications to an ntfy topic with priority, tags, title, click action, and attachment support.
- **`ntfy://{topic}` resource** — fetch recent messages for a given topic.
- **Ntfy publish service** — typed client with config, error mapping, and structured logging via the project's logger and error-handler utilities.
